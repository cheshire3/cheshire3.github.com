Cheshire3 Objects: Cheshire3 Abstract Base Class

Description

Abstract Base Class for all configurable objects within the Cheshire3 framework. It is not the base class for Document, Record, ResultSet, or Query.

Implementations

There are no pre-configured out-of-the-box ready objects of this type.

API

Module: cheshire3.configParser
Classes:

  • CaselessDictionary extends dict type
    A dictionary that is case-insensitive when searching, but also preserves the keys as inserted.
  • C3Object
    Abstract Base Class for Cheshire3 configurable objects.
